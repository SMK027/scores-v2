# Propositions d'evolution

## Priorite haute

1. Notifications temps reel web et mobile
   - Alerter quand un joueur rejoint un lobby, quand une invitation d'espace arrive, quand une partie commence, ou quand une competition est sur le point de demarrer.
   - Impact : fort sur l'engagement et l'usage des fonctions multi-joueurs.
   - Web : notifications in-app + badge dans la navbar.
   - Mobile : push notifications Expo plus fines par type d'evenement.

2. Mode hors-ligne mobile avec synchronisation
   - Permettre de creer une partie, saisir des scores et consulter les dernieres donnees sans reseau.
   - Impact : tres utile en tournoi, en bar a jeux, en association, ou en salle avec mauvaise connexion.
   - A prevoir : file d'attente locale, gestion de conflits simple, statut de synchronisation.

3. Tableau de bord d'espace personnalisable
   - Choisir quels blocs afficher en haut : dernieres parties, classement rapide, prochaine competition, joueurs les plus actifs, progression hebdo.
   - Impact : rend chaque espace plus utile au quotidien.
   - Variante simple : ordre des widgets + masquer/afficher.

4. Recherche globale plus puissante
   - Une recherche unifiee sur joueurs, parties, competitions, espaces, lobbies.
   - Ajout de filtres rapides : date, type de jeu, espace, statut.
   - Impact : tres bon gain de productivite si la base grossit.

## Priorite produit

5. Systeme de QR code
   - QR pour rejoindre un espace, accepter une invitation, ouvrir une carte membre, entrer dans un lobby.
   - Impact : excellent pour les usages physiques et evenements.
   - Mobile : scan direct depuis l'app.
   - Web : generation/affichage simple cote espace.

6. Historique joueur enrichi
   - Une vraie fiche joueur avec :
   - evolution du score dans le temps,
   - taux de victoire,
   - jeux favoris,
   - adversaires les plus frequents,
   - dernieres performances.
   - Impact : forte valeur pour la fidelisation.

7. Competitions plus avancees
   - Ajouter :
   - poules,
   - elimination directe,
   - double elimination,
   - generation automatique d'affiches/tableaux,
   - export PDF des resultats.
   - Impact : gros levier si l'application vise clubs, assos, evenements.

8. Check-in de presence avant partie ou competition
   - Les membres confirment leur presence.
   - L'organisateur voit qui est pret, absent, en retard.
   - Impact : utile avant lancement de session ou tournoi.

## Priorite UX web

9. Vue calendrier des activites
   - Regrouper parties, competitions, invitations et evenements a venir dans un calendrier mensuel ou hebdo.
   - Impact : ameliore beaucoup la lisibilite quand un espace devient actif.

10. Actions en lot
   - Selectionner plusieurs joueurs, parties ou invitations pour :
   - supprimer,
   - archiver,
   - changer un statut,
   - exporter.
   - Impact : utile pour les admins et gestionnaires.

11. Assistant de creation rapide
   - Un flux guide :
   - creer un espace,
   - ajouter des joueurs,
   - creer un type de jeu,
   - lancer la premiere partie.
   - Impact : ameliore nettement l'onboarding.

12. Centre d'activite consolide
   - Un ecran unique avec :
   - invitations,
   - lobbies ouverts,
   - parties recentes,
   - actions administratives importantes,
   - alertes de securite.
   - Impact : meilleure lisibilite globale.

## Priorite mobile

13. Saisie de score optimisee tactile
   - Grands boutons, clavier numerique adapte, gestes simples, mode arbitre.
   - Impact : tres concret sur l'usage reel mobile.

14. Carte membre mobile plus exploitable
   - Ajouter :
   - QR personnel,
   - infos d'espace,
   - role,
   - statut,
   - acces rapide a l'historique.
   - Impact : bon lien entre usage numerique et usage terrain.

15. Ecran Aujourd'hui
   - Une page mobile centree sur l'instant :
   - competitions du jour,
   - invitations recues,
   - lobbies en cours,
   - parties a reprendre,
   - rappels.
   - Impact : bon point d'entree mobile.

16. Raccourcis rapides depuis l'accueil mobile
   - Nouvelle partie, Inviter, Scanner un QR, Reprendre la derniere session.
   - Impact : fort gain de fluidite.

## Priorite admin et qualite

17. Journal d'activite filtrable et exploitable
   - Etape suivante :
   - filtres par utilisateur, action, espace, periode,
   - export CSV,
   - regroupement par type d'evenement.
   - Impact : tres utile pour audit et support.

18. Parametres d'espace plus fins
   - Regles par espace :
   - droits par role,
   - types de jeux autorises,
   - visibilite des classements,
   - invitations publiques autorisees ou non.
   - Impact : rend le produit plus adaptable.

19. Sauvegarde et export complet d'espace
   - Export JSON ou ZIP d'un espace avec joueurs, parties, competitions, reglages.
   - Impact : rassure les admins et facilite migration/sauvegarde.

20. Indicateurs de sante de l'application
   - Cote admin :
   - erreurs recentes,
   - files en attente,
   - notifications echouees,
   - uploads problematiques,
   - comptes a verifier.
   - Impact : utile si l'application continue a grossir.

## Recommandations de demarrage

1. Notifications temps reel
2. Mode hors-ligne mobile
3. QR code pour rejoindre un espace ou un lobby
4. Historique joueur enrichi
5. Tableau de bord d'espace personnalisable
